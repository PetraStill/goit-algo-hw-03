## Завдання № 1

### Як запускати скрипт

Перед запуском потрібно або:

1. **перейти у директорію, де лежить файл `task_1.py`**,  
   наприклад:
   ```bash
   cd /Users/banana/Documents/neo/algorythms/hw_2 
   ````

2. або **вказати повний шлях до скрипта** у команді:

   ```bash
   python3 /Users/banana/Documents/neo/algorythms/hw_2/task_1.py <source> <destination>
   ````

Без цього операційна система не знатиме, де шукати файл Python-скрипта.

---

### Приклади запуску

#### 1. Запуск із вказанням обох параметрів (source + destination)

У цьому випадку файли будуть переміщені до вибраної вами директорії:

```bash
python3 task_1.py "/Users/banana/Downloads/Telegram Desktop" "/Users/banana/Documents/Telegram_Data"
```

У директорії `/Users/banana/Documents/Telegram_Data` будуть створені підпапки за розширеннями файлів (`jpg`, `png`, `mp4`, `pdf`, `pfx`, тощо), і всі файли будуть переміщені відповідно до свого типу.

#### 2. Запуск тільки з директорією-джерелом (лише source)

Якщо не вказати директорію призначення, скрипт автоматично створює папку **dist** у тій директорії, *з якої було запущено скрипт*, і переміщує файли туди:

```bash
python3 task_1.py "/Users/banana/Downloads/Telegram Desktop"
```

> **Увага:** у такому випадку вихідна директорія (source) стане порожньою, тому що файли переміщуються (`shutil.move`), а не копіюються.

Приклад результату:

* файли були в `/Users/banana/Downloads/Telegram Desktop`
* скрипт був запущений з `/Users/banana/Documents/neo/algorythms/hw_2`
* отже результати перемістилися у:

```
/Users/banana/Documents/neo/algorythms/hw_2/dist/
```

Всередині `dist` будуть автоматично створені підпапки за типами файлів:

```
dist/
    jpg/
    jpeg/
    png/
    mp3/
    mp4/
    mov/
    pdf/
    docx/
    no_extension/
    ...
```

Скрипт створює ці директорії автоматично.

---

#### Пояснення параметрів

* **source** — обов’язковий параметр
  Шлях до директорії, з якої потрібно перемістити файли.

* **destination** — необов’язковий параметр
  Якщо не вказано, використовується `"dist"` у поточній директорії запуску.

## Завдання № 2
## Фрактал «Сніжинка Коха»

Програма будує фрактал «сніжинка Коха» за допомогою рекурсії та модуля `turtle`.  
Користувач вводить рівень рекурсії, після чого програма візуалізує три криві Коха, що утворюють сніжинку.

### Як запустити

```bash
python3 task_2.py
```

Після запуску програма запитає рівень рекурсії (ціле число ≥ 0).

#### Вимоги
Python 3.x  
Вбудований модуль turtle    

#### Примітки
Рівні рекурсії вище 6 можуть значно уповільнити процес малювання.
